# Лабораторные работы по дисциплине "Разработка веб-приложений на Python"

## Описание проекта

Данный репозиторий содержит выполненные лабораторные работы по курсу веб-разработки на языке Python с использованием фреймворка Django. В ходе выполнения работ были изучены основы языка Python, архитектура веб-приложений, работа с базами данных, шаблонизация, обработка форм, система авторизации и основы JavaScript.

Итоговым результатом является полнофункциональное веб-приложение "Блог" с возможностью регистрации пользователей, авторизации, создания и просмотра статей.

## Используемые технологии

- Python 3.13
- Django 6.0
- SQLite (база данных)
- HTML5, CSS3
- JavaScript (ES6)

## Содержание лабораторных работ

### Лабораторная работа №1. Основы языка программирования Python

**Цель работы:** изучить основы языка программирования Python, освоить работу с переменными, списками, словарями, функциями и циклами.

**Выполненные задания:**
- Написана программа `helloworld.py`, выводящая приветствие на экран
- Создана программа `mygroup.py` для работы со списком студентов группы
- Реализована функция `print_students()` для вывода информации о студентах
- Реализована функция `filter_by_average()` для фильтрации студентов по среднему баллу
- Создан первый Django-проект с настроенной админ-панелью

**Изученные концепции:** переменные, типы данных, списки, словари, функции, циклы for, условные операторы if/else, ввод данных от пользователя.

### Лабораторная работа №2. Создание первого веб-приложения на Django

**Цель работы:** изучить структуру Django-проекта, создать первое веб-приложение с HTML-шаблоном и CSS-стилями.

**Выполненные задания:**
- Создан Django-проект `firstwebpage` с приложением `flatpages`
- Написано представление `home()` для отображения главной страницы
- Настроена маршрутизация URL в файле `urls.py`
- Создан HTML-шаблон `index.html` с использованием таблиц и списков
- Добавлены CSS-стили для оформления страницы
- Настроена работа со статическими файлами

**Изученные концепции:** структура Django-проекта, представления (views), URL-маршрутизация, HTML-шаблоны, статические файлы, CSS-стилизация.

### Лабораторная работа №3. Работа с моделями и базой данных

**Цель работы:** изучить ORM Django, создать модель данных, выполнить миграции и настроить административную панель.

**Выполненные задания:**
- Создан проект блога с приложением `articles`
- Разработана модель `Article` с полями: заголовок, автор, текст, дата создания
- Выполнены миграции для создания таблиц в базе данных
- Настроена регистрация модели в админ-панели Django
- Добавлены тестовые статьи через административный интерфейс

**Изученные концепции:** ORM (Object-Relational Mapping), модели Django, поля моделей, миграции базы данных, административная панель Django.

### Лабораторная работа №4. Отображение данных из базы данных

**Цель работы:** создать представления и шаблоны для отображения списка статей и отдельной статьи на веб-странице.

**Выполненные задания:**
- Создано представление `archive()` для отображения списка всех статей
- Создано представление `get_article()` для отображения отдельной статьи по её идентификатору
- Настроены URL-маршруты с параметрами (`/article/<id>/`)
- Разработаны шаблоны `archive.html` и `article.html`
- Реализована обработка ошибки 404 при отсутствии статьи

**Изученные концепции:** запросы к базе данных через ORM, передача контекста в шаблоны, параметры URL, шаблонные теги Django (for, if), обработка исключений.

### Лабораторная работа №5. Создание статей через веб-форму

**Цель работы:** реализовать функционал создания новых статей через HTML-форму с обработкой POST-запросов.

**Выполненные задания:**
- Создано представление `create_post()` для обработки формы создания статьи
- Разработан шаблон `create_post.html` с HTML-формой
- Реализована проверка авторизации пользователя
- Настроена CSRF-защита формы
- Добавлен редирект на страницу созданной статьи после успешного сохранения

**Изученные концепции:** обработка GET и POST запросов, HTML-формы, CSRF-защита, создание объектов в базе данных, редиректы.

### Лабораторная работа №6. Регистрация и авторизация пользователей

**Цель работы:** реализовать систему регистрации и авторизации пользователей с использованием встроенных средств Django.

**Выполненные задания:**
- Создано представление `register()` для регистрации новых пользователей
- Создано представление `login_view()` для входа в систему
- Разработаны шаблоны `register.html` и `login.html`
- Реализована проверка уникальности логина при регистрации
- Добавлено хэширование паролей через `User.objects.create_user()`
- Настроена аутентификация через `authenticate()` и `login()`

**Изученные концепции:** модель User в Django, аутентификация и авторизация, сессии, хэширование паролей, проверка подлинности пользователя.

### Лабораторная работа №7. Основы JavaScript и работа с DOM

**Цель работы:** изучить основы языка JavaScript, освоить работу с DOM-элементами и обработку событий.

**Выполненные задания:**
- Создан файл `helloworld.js` с демонстрацией основных конструкций JavaScript
- Реализованы примеры работы с переменными, функциями, массивами, циклами
- Создан файл `fold-post.js` для интерактивного сворачивания/разворачивания статей
- Добавлена работа с DOM: поиск элементов, изменение стилей, обработка событий

**Изученные концепции:** синтаксис JavaScript, переменные (let, const), функции, массивы, DOM API, события (addEventListener), манипуляция элементами страницы.

## Инструкция по запуску

### Требования

- Python 3.10 или выше
- pip (менеджер пакетов Python)

### Установка зависимостей

```bash
pip install django
```

### Запуск блога (лабораторные работы №3-7)

```bash
cd lab3/blog
python manage.py runserver
```

После запуска сервера откройте в браузере: http://127.0.0.1:8000/

### Доступные страницы блога

| Адрес | Описание |
|-------|----------|
| `/` | Главная страница со списком всех статей |
| `/article/1/` | Просмотр статьи с указанным идентификатором |
| `/article/new/` | Форма создания новой статьи (требуется авторизация) |
| `/register/` | Страница регистрации нового пользователя |
| `/login/` | Страница входа в систему |
| `/admin/` | Административная панель Django |

### Учётные данные для входа

- Логин: `admin`
- Пароль: `admin`

### Запуск лабораторной работы №1

```bash
cd lab1
python helloworld.py
python mygroup.py
```

### Запуск лабораторной работы №2

```bash
cd lab2/firstwebpage
python manage.py runserver
```

После запуска откройте: http://127.0.0.1:8000/
